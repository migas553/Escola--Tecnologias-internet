HTML Cheat Sheet

Formulario < form >

A tag < form > é usada para criar um formulário para entrada de dados do utilizador.
O formulário pode conter vários tipos de campos de entrada, como caixas de texto, botões de opção, caixas de verificação, botões de envio, etc. A tag form tem vários atributos, como action, method, target, novalidate, autocomplete, etc.

Atributos:

Atributo Descrição
action Define para onde os dados do formulário devem ser enviados.
method Define o método HTTP para enviar os dados do formulário.
target Define onde abrir o resultado do envio do formulário.
novalidate Define que o formulário não deve ser validado quando enviado.
autocomplete Define se o preenchimento automático está ativado no formulário.
enctype Define como os dados do formulário devem ser codificados ao enviá-los para o servidor.

Fieldset < fieldset >

A tag < fieldset > é usada para agrupar elementos relacionados em um formulário.
O elemento < fieldset > desenha uma caixa ao redor dos elementos relacionados.
A tag fieldset tem vários atributos, como form, name, disabled, etc.

Atributos:

Atributo Descrição
form Define um ou mais formulários aos quais o grupo de campos pertence.
name Define o nome do grupo de campos.
disabled Define que o grupo de campos deve ser desativado.

Label < label >

A tag < label > define um rótulo para muitos elementos de formulário.
O elemento < label > não realiza nenhuma ação por si só quando clicado, mas pode ser associado a um elemento de formulário, como um campo de texto, botão de rádio, etc.
O rótulo melhora a usabilidade do formulário, permitindo que o utilizador clique no rótulo para selecionar o elemento de formulário.

Atributos:

Atributo Descrição
for Define qual elemento de formulário está associado ao rótulo.
form Define um ou mais formulários aos quais o rótulo pertence.

Input < input >

A tag < input > é usada para criar um controle interativo para formulários baseados na Web para aceitar dados do utilizador.
O atributo type especifica o tipo de elemento de formulário a ser exibido.
A tag input tem vários atributos, como type, name, value, placeholder, required, etc.

Atributos:

Atributo Descrição
type Define o tipo de controle de formulário a ser exibido.
name Define o nome do controle de formulário que é enviado para o servidor.
value Define o valor inicial do controle de formulário.
placeholder Define um texto de exemplo que é exibido no controle de formulário quando ele está vazio.
required Define que o controle de formulário deve ser preenchido antes de enviar o formulário.
pattern Define um padrão que o valor do controle de formulário deve corresponder.
maxlength Define o número máximo de caracteres que o utilizador pode inserir em um campo de texto.

Textarea < textarea >

A tag < textarea > é usada para criar um controle de formulário que permite ao utilizador inserir texto em várias linhas.
A tag textarea tem vários atributos, como name, rows, cols, form, required, etc.

Atributos:

Atributo Descrição
name Define o nome do controle de formulário que é enviado para o servidor.
rows Define o número de linhas visíveis em um controle de formulário.
cols Define o número de colunas visíveis em um controle de formulário.
form Define um ou mais formulários aos quais o controle de formulário pertence.
required Define que o controle de formulário deve ser preenchido antes de enviar o formulário.

Select < select >

A tag < select > é usada para criar um controle de lista suspensa.
A tag select tem vários atributos, como name, size, multiple, form, required, etc.

Atributos:

Atributo Descrição
name Define o nome do controle de formulário que é enviado para o servidor.
size Define o número de opções visíveis em um controle de lista suspensa.
multiple Define que o utilizador pode selecionar mais de uma opção em um controle de lista suspensa.
form Define um ou mais formulários aos quais o controle de lista suspensa pertence.
required Define que o controle de formulário deve ser preenchido antes de enviar o formulário.

Button < button >

A tag < button > define um botão clicável.
O botão pode ter conteúdo em seu interior, como texto ou imagens.
A tag button tem vários atributos, como type, name, value, form, formaction, formenctype, formmethod, formnovalidate, formtarget, etc.

Atributos:

Atributo Descrição
type Define o tipo de botão.
name Define o nome do botão que é enviado para o servidor.
value Define o valor inicial do botão.
form Define um ou mais formulários aos quais o botão pertence.
formaction Define para onde os dados do formulário devem ser enviados.
formenctype Define como os dados do formulário devem ser codificados ao enviá-los para o servidor.
formmethod Define o método HTTP para enviar os dados do formulário.
formnovalidate Define que o formulário não deve ser validado quando enviado.
formtarget Define onde abrir o resultado do envio do formulário.
