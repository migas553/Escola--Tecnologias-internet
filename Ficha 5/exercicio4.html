Principais formatações

Negrito/Destaque

Texte em negrito usando a tag bold (b) (não semantica)

Texto destacado usando a tag strong (strong) (semantica)

Italico/Enfase

Texto em italico usando a tag italic (i) (não semantica)

Texto emfase usando a tag emphasis (em) (semantica)

Texto marcado

Texto marcado usando a tag mark

agora com cor de fundo verde marcado usando a tag mark e atrbuindo um estilo inline

texto grande e pequeno

Texto grande usando a tag big

Texto pequeno usando a tag small

Texto excluido

Texto excluido usando a tag del para indicar que deve ser lido mas não é relevante

Texto inserido

Texto inserido usando a tag ins para indicar que foi adicionado depois

Existe também a tag underline mas não é semantica

Texto sobrescrito

Texto sobrescrito usando a tag sup para indicar que está acima da linha, exemplo: X2

Texto subscrito

Texto subscrito usando a tag sub para indicar que está abaixo da linha, exemplo: H2O
