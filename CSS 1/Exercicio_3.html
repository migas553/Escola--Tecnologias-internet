Cascade Style Sheets - CSS

As folhas de estilo em cascata (Cascading Style Sheets ou CSS) são um mecanismo para adicionar estilo (cores, fontes, espaçamentos, etc.) a um documento web.

O atributo 'color' em CSS é utilizado para definir a cor do texto dentro de um elemento HTML. Ele aceita uma variedade de valores, incluindo nomes de cores comuns (como 'red' para vermelho), códigos hexadecimais (por exemplo, '#FF0000' para vermelho), valores RGB (como 'rgb(255, 0, 0)' para vermelho), valores HSL (por exemplo, 'hsl(0, 100%, 50%)' para vermelho), e até mesmo valores de transparência utilizando RGBA ou HSLA. Isso permite uma ampla personalização da cor do texto em um documento web, oferecendo flexibilidade tanto para design quanto para acessibilidade.
